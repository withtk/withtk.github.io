
### 1.로컬 실행


### 2.main 병합 전 빌드
```
npm run build
```

### 3.Build flow
1. 빌드되는 디렉토리 삭제 및 하위 삭제
2. 같은 이름의 디렉토리 생성
3. 그 안에 빌드파일 넣기

실행전 TARGET_DIR_NAME(move_build_Files.js) 변경할 것.
